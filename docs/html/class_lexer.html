Подробное описание

template<class InputIterator>
class Lexer< InputIterator >

Лексический анализатор.

Формирует лексемы. Принимает интераторы отвечающие категории InputIterator, где char_type = char. Выбрасывает исключение std::invalid_argument, когда вставка элемента в keyWords, заканчивается неудачно.
